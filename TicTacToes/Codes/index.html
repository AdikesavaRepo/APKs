Tic-Tac-Toe

How to Play

Objective: Get three of your marks in a rowâ€”horizontally, vertically, or diagonally.

Starting the Game: Click the "Reset Game" button to start a new game.

Gameplay: Players take turns clicking on empty cells. The first player uses "X" and the second player uses "O".

Winning: The game checks for a winner after each turn. If a player gets three in a row, their victory message will be displayed.

Draw: If all cells are filled and no player has three in a row, the game will declare a draw.

Resetting: Click the "Reset Game" button at any time to start over.
